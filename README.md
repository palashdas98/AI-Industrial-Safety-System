# AI Industrial Safety System

## Project Overview

This project detects human entry into restricted industrial work zones using YOLOv8 and Computer Vision.

## Features

- Human Detection
- Multiple Restricted Zones
- Intrusion Detection
- Voice Warning
- Screenshot Capture
- CSV Logging
- Adjustable Zones

## Technologies

- Python
- OpenCV
- YOLOv8
- Ultralytics
- Windows Speech API
